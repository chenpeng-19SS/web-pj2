## 项目基本需求：

基本完成

### 主页
展示收藏量最多的六张图片，可以在主页实现登录登出状态的切换，点击图片跳转详情页。

### 登录页

使用用户名或者邮箱登录；错误的密码会有提示；登录后自动跳转到主页

### 注册页

两次密码输入需相同才能提交；提交后用户名或邮箱重复中止注册并有文字提示；密码弱会有文字提示

### 浏览页
实现国家与城市的筛选框二级联动；热门区点击即可筛选；
可以跳转页码。

### 搜索页

支持英文搜索，模糊查询，可以跳转页码，点击进入详情。

支持模糊查询；页码跳转
# 上传页
识别上传与修改；修改时填充待改图片的title和description，并展示待改图片；国家与城市筛选框二级联动
# 库页面
无结果时有文字提示；修改按钮可以跳转到上传页；删除按钮删除照片；页码跳转
# 收藏页
### 详情页
收藏按钮内文字切换；未登录时收藏按钮引导至登录页

### 上传页面

可以进行上传，并且必须进行信息补全。

### 我的照片

显示自己上传的照片，可以删除照片，可以跳转页码。

### 我的收藏

显示自己收藏的照片，可以删除收藏，可以跳转页码
